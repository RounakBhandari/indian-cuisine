# 🍽 Indian Cuisine — Interactive Restaurant Website

An immersive and visually engaging restaurant website built with **React**, **Vite**, **Tailwind CSS**, and **Framer Motion**.

This project is designed to present a restaurant’s **cuisine, ambience, and story** through cinematic layouts, smooth animations, and modern UI/UX interactions. The goal is to create a digital experience that reflects the **luxury, culture, and authenticity** of Indian and Nepali food.

The website is being developed for **Indian Cuisine**, a restaurant located in **Austria**, run by **Chef Madhukar Bhandari**, who has over **20 years of culinary experience** working in Nepal, England, and Vienna.

---

# 🌍 Project Vision

Most restaurant websites are simple and static.
This project aims to go beyond that by building a **modern, interactive, and emotionally engaging website**.

The design focuses on:

* Cinematic scrolling experiences
* Smooth animations and transitions
* Luxury themed design
* Interactive UI elements
* Story-driven restaurant branding

The overall aesthetic uses a **gold, black, and white theme** inspired by luxury fine-dining restaurants.

# ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/your-username/indian-cuisine-website.git
```

### 2️⃣ Navigate into the project folder

```
cd indian-cuisine-website
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Run the development server

```
npm run dev
```

The project will start on:

```
http://localhost:5173
```
